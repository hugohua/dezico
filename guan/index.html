北京 百视文化传媒有限公司（简称百视传媒），注册资本2000万元。

百视传媒主营业务包括动画片创作与制作、动漫游戏设计开发、动漫品牌授权、动漫产品发行、图书音像制品制作发行、影视剧投资、电视节目制作等。

百视传媒与北京电视台卡酷少儿和北京文化硅谷联合出品原创系列TV动画片《我的朋友猪迪克》，并推出DEZICO动漫品牌。于2014年4月推出猪迪克在线教育识字软件。

PASS MEDIA (Beijing) CO., LTD (refered to as PASS MEDIA) is registered with 3,5 million dollar capital.

The main businesses of PASS MEDIA include cartoon investment and production, animated game design and publishing, animation brand licensing, animation products publication, book and audio-visual production and distribution, television series investment, television program production etc.

In 2013, PASS MEDIA has produced the original TV cartoon series My friend DEZICO, and launched DEZICO cartoon brand, which signifies that PASS MEDIA has officially entered the field of brand licensing. In 2014 PASS MEDIA will launch the first appliction in App Store "DEZICO literacy".
